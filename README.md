# Train_

A personal CrossFit return-to-fitness PWA.

## Use on iPhone
- Enable GitHub Pages (Settings → Pages → main)
- Open URL in Safari → Share → Add to Home Screen

## What it does
- Day-by-day sessions w/ warm-up, strength, WOD, cool-down
- Pairing picker, custom workout builder
- Session player w/ EMOM/AMRAP/RFT timers
- 1RM-driven weights, per-movement scale levels
- localStorage-only, offline-capable
